## Rodar o trabalho:

<p>
    Clone o projeto
</p>
```
git clone https://github.com/Andreabenf/FSE_2022_2_TRAB2.git
```
<p>
    Entre na pasta do projeto
</p>
```
cd FSE_2022_2_TRAB2
```
<p>
    Crie uma paste bin (caso não crie, o makefile pode dar erro)
</p>
```
mkdir bin
```
<p>
    Compile o projeto:
</p>
```
make
```
<p>
    Rode o projeto:
</p>
```
bin/bin
```
